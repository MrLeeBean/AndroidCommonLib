# AndroidCommonLib
Android快速开发公共库


Step 1：Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}

Step 2. Add the dependency

	dependencies {
	         implementation 'com.github.changshuai7:AndroidCommonLib:1.0.1'
	}
